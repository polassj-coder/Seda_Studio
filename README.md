# Seda Studio - Landing Page Premium 💎

Una landing page de alto impacto diseñada para una agencia de desarrollo y diseño web. Este proyecto está estructurado para maximizar las conversiones mediante el uso de neuromarketing, un diseño lujoso (Dark Mode + Dorado Obsidiana) y una herramienta interactiva única.

## 🚀 Características Principales

*   **Diseño Premium (Dark Mode):** Estética visual basada en colores oscuros profundos y acentos dorados para transmitir autoridad, confianza y exclusividad.
*   **Simulador Interactivo "Crea tu Visión":** Una herramienta construida en Vanilla JavaScript que permite a los clientes potenciales jugar con el aspecto de su futura web antes de comprar. 
    *   Personalización en tiempo real del **Nombre de Marca**.
    *   Cambio de **Paleta de Colores** (6 opciones incluyendo Dorado, Zafiro, Rubí).
    *   Selector de **Industria** (Actualiza la iconografía dinámicamente).
    *   Selector de **Estructura Visual** (Centrado o Dividido).
    *   Selector de **Tono de Mensaje** (Modifica el copywriting del simulador).
*   **Copywriting Orientado a Ventas:** Textos persuasivos que resaltan el valor del servicio (velocidad, conversiones y SEO) en lugar de solo características técnicas.
*   **Integración Fluida con WhatsApp:** Todos los botones de llamada a la acción (CTAs) generan un enlace directo a WhatsApp con mensajes pre-llenados (incluso dinámicos desde el simulador).
*   **100% Responsivo:** Adaptabilidad perfecta en dispositivos móviles, tablets y computadoras de escritorio.

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica.
*   **CSS3:** Estilos personalizados, CSS Variables (`:root`), Flexbox, CSS Grid y animaciones suaves (glassmorphism, micro-interacciones).
*   **Vanilla JavaScript (ES6+):** Lógica del simulador interactivo, Intersection Observers para animaciones al hacer scroll y seguimiento del cursor (blobs).
*   **FontAwesome:** Iconografía vectorial.
*   **Google Fonts:** Tipografías modernas (*Outfit, Playfair Display, Space Grotesk*).

## 📂 Estructura del Proyecto

## ⚙️ Uso

Este proyecto no requiere procesos de compilación (build tools) ni dependencias de Node.js. Para visualizar la página:

1. Clona el repositorio.
2. Abre el archivo `index.html` en cualquier navegador web moderno (Google Chrome, Firefox, Safari, Edge).
